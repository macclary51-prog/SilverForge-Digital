Support

Help with SilverForge apps and services

Find general troubleshooting information or contact SilverForge about an application, website or active service.

Before requesting support

For an application issue, try these steps first:

  • Close and reopen the application.
  • Confirm the application is updated to the newest available version.
  • Restart the device if the problem continues.
  • Write down what happened immediately before the issue.
  • Take a screenshot if an error message appears.

Information to include

When requesting help, include:

  • The name of the application or service.
  • Your device type and operating-system version.
  • The application version, if available.
  • A clear description of the problem.
  • The steps that cause the problem to happen.

Website and business-service support

For an active website, social media or content service, describe the requested change and identify any affected page, account, post or campaign.

Contact support

Use the contact page and select the closest service option. Include “Support” at the beginning of the project-details message.
